电脑网站支付
-------------------------
	# 场景
		* 在PC网站点击付款后, 跳转到支付宝的支付页面
		* 可以使用APP扫描二维码付款
		* 也可以登录后, 使用支付密码支付
	
	# 文档
		https://docs.open.alipay.com/270
## 手机网站支付
	# 场景
		* 在移动端网页应用中集成支付宝
		
		* 在手机网页中唤醒支付宝APP进行支付, 支付完后跳回到自己网页
		* 如果唤醒失败(可能没安装), 在一定的时间后会自动进入网页支付流程
	
	# 文档
		https://docs.open.alipay.com/203
APP支付
----------------------
	# 场景
		* 自己开发的APP
		* 在APP里面唤醒支付宝APP进行支付
		* 支付完后跳回到自己APP内, 最后展示支付结果
		
		* 如果用户没有安装支付宝 APP, 商家 APP 内会调起支付宝网页支付收银台
		* 用户登录支付宝账户, 支付完后展示支付结果
	
	# 文档
		https://docs.open.alipay.com/204/
